# MVP 产品需求文档

## 用户

首版用户：准备开一家小店的创业者，例如奶茶店、美甲店、火锅店、咖啡店、小吃店。

## 核心场景

用户已经有一个候选地址，希望快速判断这个位置是否适合某个品类。

## 输入字段

| 字段 | 必填 | 说明 |
|---|---|---|
| 地址或经纬度 | 是 | 可先支持文本地址，后续接地图选点 |
| 城市 | 是 | 用于减少地址解析歧义 |
| 开店品类 | 是 | 例如奶茶、美甲、火锅、咖啡 |
| 月租金 | 否 | 用于成本压力估算 |
| 预算 | 否 | 用于判断资金压力 |
| 预计客单价 | 否 | 后续用于营业额估算 |

## 输出模块

| 模块 | 内容 |
|---|---|
| 一句话结论 | 推荐、谨慎推荐、高风险、不建议 |
| 总评分 | 0-100 分 |
| 分项评分 | 客流潜力、竞争压力、消费匹配、成本压力、交通可达、经营风险 |
| 周边摘要 | 竞品数量、住宅/办公/交通/商圈数量 |
| 风险提示 | 数据不足、竞争过高、成本过高等 |
| 下一步建议 | 实地踩点、竞品观察、租金核实、访谈等 |
