# 项目简报

## 项目一句话

为准备开店或正在经营门店的人提供选址可行性评估和经营策略模拟：输入位置和品类，系统输出评分、风险、原因和下一步建议。

## MVP 范围

首版只做一个闭环：

1. 用户输入地址或经纬度。
2. 用户选择开店品类。
3. 后端获取周边 POI 数据。
4. 后端计算基础评分。
5. 前端展示可读报告。

## 暂不做

- 账号体系和支付。
- 自动抓取美团/大众点评。
- 复杂机器学习训练。
- 直接把 MiroFish 代码混进主项目。
- 对开店成功率作确定性承诺。

## 成功标准

普通用户输入一个真实地址后，3 分钟内得到一份能看懂的报告。报告应包含结论、分项评分、关键依据、风险提示和下一步行动。
