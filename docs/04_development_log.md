# 开发日志

## 2026-06-10

- 检查开发环境。
- 初始化 Git 仓库和项目结构。
- 建立前端、后端、文档、MiroFish 实验区。
- 确认本机 Python 3.12.0 可用，并创建 `backend/.venv`。
- 安装后端依赖，健康检查接口和分析接口通过冒烟测试。
- 接入高德 Web 服务：地理编码和周边 POI 搜索。
- 新增 `/api/pois/search` POI 预览接口。
- 升级 `/api/analysis`：未传手工 POI 统计时自动调用高德采集，并将 POI 汇总纳入评分。
- 优化 POI 采集方式：按类型分组搜索、去重、分类汇总，并加入高德 QPS 限速重试。
- 完成前端联调：输入城市、地址、品类、租金和预算后，可在网页生成真实 POI 评分报告。
- 浏览器验证结果：广州天河区体育西路、奶茶店、1000 米半径，返回 151 条去重 POI，评分 76，等级为“谨慎推荐”。
- 接入地图选点：后端新增 `/api/map/geocode` 和 `/api/map/static`，前端可先按文字地址定位，再点击静态地图调整经纬度，并用选点坐标生成分析报告。
- 浏览器验证结果：广州天河区体育西路定位到 `113.321570,23.133440`，点击地图后更新为 `113.323204,23.132936`，生成报告评分 80，等级为“推荐”。
- 优化地图选点流程：前端改为城市 + 区域/商圈/街道先定位范围，点击地图后切换为精确点；详细位置作为备注保留，不再打断已选坐标。
- 开店品类由固定下拉改为自由输入，并保留常用品类候选。
- 浏览器验证结果：上海浦东新区陆家嘴定位到 `121.507513,31.234295`，点击地图后更新为 `121.510727,31.233872`，自定义品类“潮玩集合店”生成报告评分 92，等级为“推荐”。
- 升级地图拾取器：在不暴露高德 Key 的前提下，前端支持地图拖拽平移、放大缩小、点击定点和“搜索详细”候选定位，体验更接近坐标拾取工具。
- 浏览器验证结果：广州体育西路范围定位后，放大、拖拽和点击均能更新坐标；点击后的 `113.316351,23.134703` 可成功生成报告。
- 后端验证结果：深圳南山区科技园万象天地可解析到 `113.954221,22.541787`，地图坐标分析接口可正常生成轻食沙拉店报告。
- 简化地图拾取器：移除“定位范围”按钮，只保留“搜索详细”作为定位入口；搜索结果显示为候选点，拖拽或点击地图后才切换为精确点。
- 浏览器验证结果：页面仅保留“搜索详细”和“清除”按钮，广州体育西路搜索后点击地图更新到 `113.322051,23.133314`，可成功生成报告。
- 完成第2周低保真交付物：新增简短 PRD、页面流程图和可点击 HTML 原型，覆盖输入页、加载页、报告页和历史页。
- 原型验证结果：输入页搜索详细和地图点击可更新坐标，加载页包含商圈、竞品、交通、住宅进度，失败态包含重试按钮，报告页可保存到本地历史。
- 修复地图详细位置跳转不准：新增 `/api/map/search` 真实 POI 候选搜索，前端“搜索详细”显示候选列表，用户选择候选后地图跳到该真实地点坐标。
