# Frontend

前端当前是一个轻量 Vite 单页应用，用于输入候选城市、区域范围、详细位置、开店品类、租金和预算，并调用后端生成真实 POI 分析报告。

## 启动

```powershell
cd frontend
npm run dev
```

默认地址：

```text
http://127.0.0.1:5173
```

## 当前能力

- 输入候选城市、区域/商圈/街道和详细位置。
- 可通过“搜索详细”返回真实 POI 候选点，选择候选后再通过拖拽、缩放、点击地图确认精确经纬度。
- 自由填写开店品类，并选择分析半径。
- 调用后端 `/api/analysis`。
- 展示综合评分、推荐等级、分项评分、POI 汇总和可点击分类 POI 明细。
